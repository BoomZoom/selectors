Les sélecteurs CSS

Voici un exercice un peu plus interractif et proche de la réalité en HTML/CSS pour utiliser les sélecteurs.

  1. Choisis une page d'exercice dans la liste ci-dessous.
  2. Ouvre le fichier CSS et HTML correspondant à l'exercice
  3. Ajoute le(s) sélecteur(s) CSS correcte(s) au fichier CSS de l'exercice pour que la consigne soit respecté.
  4. Consulte la page pour voir si le résultat est bon ou non.

Tu ne dois rien modifier comme propriétés CSS ou l'HTML! Tu dois juste écrire le sélecteur correctement pour que la consigne soit respectée!
